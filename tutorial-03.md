# Python Tutorial for Beginners 3: Integers and Floats - Working with Numeric Data
[Video](https://www.youtube.com/watch?v=khKv-8q7YmY)

## Arithmetic Operators:
### Addition:
    3 + 2
### Subtraction:
    3 - 2
### Multiplication:
    3 * 2
### Division:
    3 / 2
### Floor Division:
    3 // 2
### Exponent:
    3 ** 2
### Modulus:
    3 % 2

## Comparisons:
### Equal:
    3 == 2
### Not Equal:
    3 != 2
### Greater Than:
    3 > 2
### Less Than:
    3 < 2
### Greater or Equal:
    3 >= 2
### Less or Equal:
    3 <= 2

## Round number to one decimal
    round(3.14, 1)
